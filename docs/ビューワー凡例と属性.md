# 05_ポリゴン表示ビューワー：凡例・場所・属性の地域名

## サーバー起動

```bash
cd MapLibre-HandsOn-Beginner/05_ポリゴン表示
python3 serve.py
```

ブラウザで **http://localhost:8080/** を開く（WSL2 の場合は Windows から localhost:8080 でアクセス可能な場合あり）。

## 属性情報に表示される「地域名」の出どころ

- **工業用地**: 属性 `L05_002` をポップアップで表示（地名・用途名など）。
- **管理三角点**: 名称(meisyo)、所在地(syozaiti)、基準点コード(kijyunten_cd)、緯度b・経度l など。地域名は **所在地** など。
- **14条地図**: ポップアップタイトル「14条完了エリア」。属性は **ID**, **AREA**。地域名は AREA に含まれる場合あり。
- **土地活用推進調査**: パネル「土地活用推進調査」で表示。**市区町村名**（col2）、**住所等**（col11）、座標・市町村コードなど。地域名は **市区町村名**・**住所等**。
- **街区基準点等**: ポップアップで **市区町名**・**所在地**・基準点コード・街区点・補助点名称・標高・測量年月日 など。地域名は **市区町名**・**所在地**。
- **都市部官民基準点等（TKS_04206）**: ポップアップで **市区町名**・**所在地**・**基準点等名称**・基準点コード・座標系・標高・測量年月日 など。地域名は **市区町名**・**所在地**（白石市）。
- **都市部官民基準点等（マージ）**: ポップアップで **市区町名**・**所在地**・**基準点等名称**・基準点コード・座標系・標高・測量年月日 など。地域名は **市区町名**・**所在地**。
